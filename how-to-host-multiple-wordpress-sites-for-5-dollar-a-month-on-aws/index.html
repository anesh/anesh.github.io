How to host multiple wordpress sites for $5/month on AWS

Anesh Kesavan

Oct 16, 2021 • 2 min read

The answer is to deploy a LAMP instance on Amazon lighsail for $5 and configuring each new wordpress website as a virtual host in Apache

Lets start with Creating the Virtual Host config in Apache

  1. Login via SSH using the SSH key pair and change directory to the below path
cd /opt/bitnami/apache/conf/vhosts

2. create two files like below ideally with the name of your website, to distinguish between each wordpress site, here we are using "example" as the name of the wordpress site.

example.conf, below is the vhost configuration for http

<VirtualHost *:80>
  ServerName example.com
  ServerAlias www.example.com
  DocumentRoot /opt/bitnami/example
  RewriteEngine On
  RewriteCond %{HTTPS} !=on
  RewriteCond %{HTTP_HOST} !^(localhost|127.0.0.1)
  RewriteRule ^/(.*) https://%{SERVER_NAME}/$1 [R,L]

  <Directory "/opt/bitnami/example">
    Options -Indexes +FollowSymLinks -MultiViews
    AllowOverride All
    Require all granted
  </Directory>
</VirtualHost>

example-https.conf, below is the vhost configuration for https

<VirtualHost *:443>
  ServerName example.com
  ServerAlias www.example.com
  SSLEngine on
  SSLCertificateFile "/opt/bitnami/apache/conf/example.com.crt"
  SSLCertificateKeyFile "/opt/bitnami/apache/conf/example.com.key"
  DocumentRoot /opt/bitnami/example
  <Directory "/opt/bitnami/example">
    Options -Indexes +FollowSymLinks -MultiViews
    AllowOverride All
    Require all granted
  </Directory>
</VirtualHost>

3. Next step is to install wordpress

create a directory with the right permissions

a)sudo mkdir /opt/bitnami/example
b)sudo chown -R bitnami:daemon /opt/bitnami/example
c)sudo chmod -R g+w /opt/bitnami/example

download wordpress and uncompress the tar file

wget https://wordpress.org/latest.tar.gz
tar -xzvf latest.tar.gz

copy the contents of the wordpress folder to the installation directory

cd wordpress
cp -r ../.

remove the uncompressed wordpress folder and the wordpress tar [below is an optional step], i like cleaning up!!

cd /opt/bitnami/example
rm -fr wordpress/
rm -fr latest.tar.gz

create a database for the wordpress site, replace username and password with your choices

mysql -u root -p
#the above command will prompt you for a password, you can get it through the below command
cat ~/bitnami_application_password

MariaDB [(none)]> CREATE DATABASE example;
Query OK, 1 row affected (0.001 sec)

MariaDB [(none)]> grant all privileges on example.* TO 'username'@'localhost' identified by 'password';
Query OK, 0 rows affected (0.004 sec)

MariaDB [(none)]> FLUSH PRIVILEGES;
Query OK, 0 rows affected (0.002 sec)

MariaDB [(none)]> exit
Bye

Open in the url http://example.com/wp-admin/install.php in your web browser

and provide the database name and credentials created above and you should have a working installation of wordpress

Repeat Steps 1 to 3. to install other wordpress sites

After launching your wordpress site,in a few months you might notice that you want to switch to a better plan or server, due to increased workload. You can find how to do that in this blog post
